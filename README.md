### Features

1. Play and control any track from your Spotify playlists
   - Play/Pause
   - Skip back/Skip next (15 secs)
2. Recognize faces, detect mood and play the track that matches it most
3. Authentication
   - OAuth2 from Spotify
4. Search for tracks
